## 名称释义

**eBarter**：
- **e** = electronic（电子）
- **barter** = 易货

Ebarter即为电子商务时代（e时代）的现代易货交易。

---

## 平台简介

易巴特是由青岛市易货交易中心投资建立的基于现代易货的大型电子商务平台，为企业提供：

- 易货信息发布、查询
- 易货交易结算
- 交易安全监控

等专业化的服务内容，帮助企业开展多边易货交易，实现：

| 功能 | 说明 |
|------|------|
| 以物易物 | 用闲置物品交换所需物品 |
| 以物抵债 | 用实物偿还债务 |
| 以物换股 | 用实物资产换取股权 |
| 盘活资产 | 激活沉睡资产的价值 |

解决企业资金短缺、产能过剩、销售不畅、积压库存、三角债等难题，促进企业产品流通和社会经济发展。

## 历史大事记

### 2002年
- 青岛市易货交易中心成立
- 由青岛市易货交易中心投巨资开通中国企业易货网
- 注册中国现代易货行业第一个电子商务服务品牌——"易巴特"

### 2003年
- 在青岛市政府的协助下，"易巴特"组织筹办了中国首届现代易货研讨会
- 这是现代易货概念引入中国以来规模最大的一次高层次研讨会
- 代表着现代易货在中国实践的最新成果

### 2004年
- 青岛市易货交易行业经青岛市相关机构批准成立首家易货商会
- 由"易巴特"联合同行组织召开了首届中国易货行业峰会
- 成立了中国易货联盟
- 峰会旨在建立现代易货行业的合作对话机制，统一现代易货行业的信息和结算标准

### 2005年
- "易巴特基于现代易货的电子商务交易平台项目"被国家发改委列为电子商务专项重点项目
- 得到专项资金的支持
- 入选国家教育部电子商务教材20家典型案例

### 2006年
- 易巴特入选《中国电子商务》杂志发起的"行走的榜样"评选活动
- 成为新生电子商务企业的代表之一
- 被中国《互联网周刊》评选为"创新营销优秀案例"
- 荣获"山东省电子商务示范单位"称号
- **福布斯杂志2006年9月期《物物交换新时代》详尽报道**
  - "现代易货通过互联网让物物交换这个古老的生意焕发出了新生机"

### 2007年
- 中国电子商务协会授权易巴特为中国易货行业协会
- 主要业务范围包括：
  - 对国内易货机构进行业务指导
  - 制定行业标准
  - 开展国际交流及人才培训
  - 咨询服务等
- 协会可以协助政府部门承担行业准入资格审核，提供机构信用证明等工作
- 中国电子商务论坛首次明确指出未来电子商务交易的三个方向之一是：现代易货交易体系
- 清华大学作为国家电子商务模型课题的承担单位派专人到中国企业易货网进行研究

### 2008年
- 易巴特与华安保险等金融机构合作
- 在学习其他金融行业的管理结构的同时不断完善自己
- 网站第四次升级，完善呼叫中心、短信平台

### 2009年
- 易巴特与建设银行合作发行了中国第一张集现金、巴特币为一体的双账户银行卡
- 开发了EPOS系统，实现了易货交易银行卡结算
- 易巴特与山东人民广播电台合作服务于广大换客朋友
- 为他们免费提供换物信息查询、交流的平台

## 易巴特联名卡

2009年5月，易巴特与中国建设银行合作发行了建行"易巴特联名卡"：

- **国内易货企业中第一个与银行联名发行的易货结算卡**
- **全国第一个可以在ATM机上使用的易货结算卡**

### 产品功能
- 集现金、巴特币为一体的双账户
- 支持ATM取款
- EPOS系统结算
